# Personal Portfolio Website (Azure Static Website)

A responsive personal portfolio website hosted using **Microsoft Azure Blob Storage Static Website Hosting**.

## Live Demo

Azure:
https://mbzcloudportfolio.z1.web.core.windows.net/

Vercel:
https://YOURPROJECT.vercel.app

## Features

- Responsive Design
- Professional Portfolio
- Experience Timeline
- Technical Skills
- Projects Section
- Resume Download
- Contact Information
- Custom 404 Page
- Hosted on Microsoft Azure

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Microsoft Azure Blob Storage
- Azure Static Website Hosting

## Project Objective

This project was completed as part of my **Cloud Computing (AWS/Azure) Internship at DecodeLabs**, demonstrating how to deploy a static website using cloud storage without provisioning a server.
